## Project Name : Digitools Platform

Description: 

Digitools Platform is a simple web application where users can explore different digital tools and services. Users can view tool details, features, and pricing plans. The project is built with React and styled using Tailwind CSS and DaisyUI to create a clean and responsive user interface.

## Technologies Used
React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
React-Toastify (NPM Package)
JSON (for product data)

## Features
1. Users can view different digital tools with pricing and features.
2. Toast notifications are shown when users interact with the platform.
3. Fully responsive design for mobile, tablet, and desktop devices.
